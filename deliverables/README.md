# Deliverables — Shared Task Outputs

Agents place completed task outputs here for cross-team access.

## Structure

```
deliverables/
├── {date}_{agent}_{topic}.md     — task result file
└── README.md                     — this file
```

## Rules

- File naming: `YYYYMMDD_agent_topic.md` (e.g. `20260318_alice_code_review.md`)
- Short-lived: move to project docs after acceptance
- NOT for long-term storage — use agent workspace-specific dirs
- AMB notification: after placing file, send `"Deliverable: deliverables/file.md"` via AMB
